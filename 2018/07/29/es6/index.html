块作用域

js中作用域有：全局作用域、函数作用域。ES6新增块作用域，由{}包括;

let定义变量

var定义的变量，没有块的概念，可以跨块访问，不能跨函数访问;
let定义的变量，只能在块作用域里访问，不能跨块访问，也不能跨函数访问；（不像var存在变量提升)

const命令

const用来定义常量，使用时必须初始化（即赋值），也只能在块作用域里访问，而且不能修改.

解构赋值

数组
变量的解构赋值 var [a, b, c] = [1, 2, 3]; // a = 1, b = 2, c = 3
var [a, , c] = [1, 2, 3]; // a = 1, c = 3
var [a, …c] = [1, 2, 3]; // a = 1, c = [2, 3]
var [a, b = 1, c = 3] = [1, 2]; // a = 1, b = 2, c = 3;
对象
var obj = {
a: 1,
b: 2
}
let {a, b} = obj;// 对象中的成员是根据变量名来赋值的
let {c, b} = obj;// c = undefined
let {a:A, b}; // 将对象中的a改名为A
解构语句{}不能放在句首，会被当成代码块，要加上()
实际上是 obj -> a 去读的，若找不到该成员，按照对象成员的访问方式会自己创建这个成员

字符串方法

  • 原方法indexOf：
    • 找出下标，找不到则为-1
  • 新方法includes
    • 直接判断是否存在该字符串
  • startsWith： 判断开头
  • endsWith： 判断结尾
  • repeat
    • 用法: string.repeat(num)
    • 重复n次字符串

模板字符串

相当于插入了html代码,优点在于断行都会保留,模板嵌套

Symbol数据类型

原有数据类型：undefined,null,Bollean,String,Number,Object
唯一标志:不会有相同的Symbol,用作key,模块化编程中需要
let a = Symbol();
let b = Symbol();
return a === b; // false

Proxy

对象方法属性化
var user = new Proxy({}, {
get: function(obj, prop) {
if (prop == ‘full_name’){
return obj.fname + ‘ ‘ + obj.lname;
}
}
})

user.fanme = ‘Bob’;
user.lname = ‘Wood’;
console.log(“user.full_name:”, user.full_name); // user.full_name: Bob Wood
console.log(“user.age:”, user.age); // user.age: undefined
