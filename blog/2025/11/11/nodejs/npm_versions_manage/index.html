Install specific version of Node.js

Uninstall versions

Switch Version

Assign specific version for different projects (Linux/macOS)

Create and edit a file named .nvmrc in the root path of projects

run nvm use in the root path, and it will use the version specified in the nvmrc file automatically.

using VSCode Plugin vsc-nvm (Linux/macOS)

Using vsc-nvm version in nvmrc automatically whenever you open the project in VSCode.

using VSCode Plugin Node Visual Manager (Windows)

Using Node Visual Manager to switch version, or manually using nvm use <version> on Windows system.
