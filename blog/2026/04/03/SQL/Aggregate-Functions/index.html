Aggregate Functions, GROUP BY, HAVING

Aggregate functions

  • avg, min, max, sum, count
  • count(*) counts rows, including those with NULLs; count(expr) counts non-NULL expr values.

GROUP BY and HAVING

  • GROUP BY partitions rows; SELECT may include only grouped columns and aggregates.
  • WHERE filters rows before grouping; HAVING filters groups after aggregation.

Common mistake: selecting non-grouped, non-aggregated columns (e.g., dept_name, ID, avg(salary) with group by dept_name) is invalid.
