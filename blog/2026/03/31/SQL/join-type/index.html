Join types in SQL
2026-03-31 19:53   label SQL

Join way

Join Types + Join Condition

Join Types

  • inner join
  • outer join
    • left
    • right
    • full

Join conditions

  • natural
  • on
  • using (A1, A2, …, An)

natural join

must have same values on all common attributes

inner join

“using”: to specific which columns should be equated “on”: allows a general predicate over the relations being joined

outer join

uses null values

left outer join

left <- right, all left rows remained

right outer join

left -> right, all right rows remained

full outer join

all left and right rows remained, unmatch rows use null values
