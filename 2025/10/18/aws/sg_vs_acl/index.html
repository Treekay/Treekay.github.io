Security Group and NACL
2025-10-18 11:26   label AWS

Security Group

  • Associated with instance
  • Stateful: allow return traffic
  • default rule:
    • allow all out-bound traffic
    • deny all in-bound traffic
  • Only allow traffic defined by explicit rules
  • Use cases: controlling access to specific instances (application access)

Network ACL

  • Associated with subnets
  • Stateless: not allow return traffic
  • Default ACL for VPC
    • allow all in-bound and out-bound traffic
  • Self-defined ACL default:
    • deny all in-bound and out-bound traffic
  • Use cases: controlling traffic bound of the subnets (network protecting, compliance, traffic control, IP blocking)
